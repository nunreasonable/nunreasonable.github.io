1. Introduction

The Cornwall Discord Bot ("the Bot") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, store, and protect your information when you use our Discord bot.

2. Information We Collect

2.1 Automatically Collected Information
  • Discord User ID
  • Discord Username
  • Server Information
  • Message Content
  • Command Usage Data
2.2 Information You Provide
  • ROBLOX Username
  • User Mentions
  • Custom Messages
2.3 Technical Information
  • Timestamps
  • Message Metadata
  • Error Logs

3. How We Use Your Information

3.1 Core Bot Functionality

  • Command Execution
  • User Verification
  • Message Management
  • Role Management

3.2 Service Improvement

  • Error Diagnosis
  • Usage Analytics
  • Security Monitoring

3.3 Communication

  • Direct Messages
  • Deployment Notifications
  • System Updates

4. Data Storage and Retention

4.1 Message Storage
  • Messages are stored in memory for a maximum of 24 hours.
  • Storage exists solely for the message reposting feature.
  • The Bot does not keep a persistent database of messages.
  • Temporary storage is cleaned up automatically.
4.2 User Data
  • Discord information is kept only as long as necessary.
  • ROBLOX verification data is retained for roster management.
  • Command logs are stored temporarily for debugging and security.
4.3 Data Minimization
  • We collect only what is necessary for Bot functionality.
  • No unnecessary personal data is collected or retained.
  • Message content is not permanently stored.

5. Data Sharing and Disclosure

No third-party sharingWe do not sell, rent, or share personal information for advertising or commercial purposes.
Limited disclosureDisclosure may occur only for legal, security, or service-provider reasons.

5.3 Regimental Management

  • ROBLOX verification information may be used for roster management.
  • User role assignments may be visible to server administrators with appropriate permissions.

6. Data Security

6.1 Technical Measures

  • Encryption through Discord's secure connections.
  • Access controls for Bot systems and data.
  • Memory management for temporary message storage.
  • Secure logging without sensitive information exposure.

6.2 Administrative Measures

  • Limited access for authorized administrators only.
  • Regular security reviews and data handling checks.
  • Training on privacy and security best practices.

7. Your Rights and Choices

7.1 Data Access

You can request information about what data we have about you by contacting the Bot administrators.

7.2 Data Correction

You can request updates to incorrect ROBLOX or other information through the administrators.

7.3 Message Control

Messages are removed automatically after 24 hours, and manual deletion requests can be made through administrators.

7.4 Bot Usage

You can stop using the Bot at any time and request removal from regimental systems if applicable.

8. Message Reposting Privacy

8.1 How It Works

  • Messages are temporarily stored for automatic reposting functionality.
  • Reposting occurs randomly in designated channels.
  • Bot messages and system messages are automatically excluded.

8.2 Your Control

  • Messages are automatically removed from storage after 24 hours.
  • You can request immediate deletion of your stored messages.
  • Server administrators can disable the reposting feature.

8.3 Limitations

  • Once reposted, messages become part of Discord's permanent chat history.
  • The Bot cannot delete messages from Discord's servers.
  • Original messages remain in Discord regardless of Bot storage.

9. Children's Privacy

The Bot is not directed to children under 13. We do not knowingly collect personal information from children under 13 and will delete it promptly if discovered.

10. International Data Transfers

Your information may be processed and stored on servers in different countries. When we transfer your data internationally, we ensure appropriate safeguards are in place.

11. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Changes will be posted here with an updated "Last Updated" date, and significant changes will be announced through official Discord channels.

12. Contact Information

If you have questions about this Privacy Policy or want to exercise your data rights, contact the Bot administrators through the official Discord server channels.

13. Discord's Privacy Policy

This Privacy Policy supplements Discord's Privacy Policy. Discord's collection and use of your data is governed by Discord's own privacy policies.
