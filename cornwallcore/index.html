Cornwall Discord Bot - Documentation

Discord Bot for the 32nd "The Cornwall" Regiment of Foot with enlistment, deployment and messaging features.

Bot Commands

Enlistment & Verification

  • /enlistuser - Enlists a user with automatic ROBLOX verification
    • Requires: Permission role configured
    • Parameters: user, ROBLOX username
    • Function: Anti-ALT verification + role assignment + nickname + log
  • /alistar-se - Self-enlistment with interactive form
    • Specific channel required
    • Function: Complete form + automatic ROBLOX verification

Spreadsheet & Information

  • /checkspreadsheetinfo - Queries regimental spreadsheet information
    • Parameter: username in spreadsheet
    • Function: Searches data in spreadsheet Roster tab

Messages & Communications

  • /deployment - Sends deployment message
    • Requires: Permission role configured
    • Parameter: game code
    • Function: Embed with roles ping + access buttons
  • /dmrole - Sends DM to role/user (with code)
    • Requires: Permission role configured
    • Parameters: role/user, code, message
    • Limit: 500 members per execution
  • /dmrolemsg - Sends DM to role/user (without code)
    • Requires: Permission role configured
    • Parameters: role/user, message
    • Limit: 500 members per execution

Repost System

  • /messages - Checks message storage status
    • Function: System statistics and progress
  • /repost - Reposts random stored message
    • Requires: Minimum of stored messages
    • Cooldown: 2 hours after manual use

Utilities

  • /ping - Tests bot latency
    • Function: Returns response time in ms

Administration Commands

  • vsfdliliane - Special restricted command
    • Requires: Specific user ID
    • Function: Custom responses

Configuration

The bot uses the config.json file for configurations of:

  • Channel and role IDs
  • Spreadsheet URLs and links
  • Command permissions
  • Custom messages
  • Blocked terms list and automatic blacklist response

Note: the bot now uses the MessageContent intent to read message text and detect blocked terms. This intent also needs to be enabled in the Discord application panel.

Requirements

  • .NET 8.0+
  • DisCatSharp
  • Proper configuration in config.json
  • Bot permissions in the Discord server
